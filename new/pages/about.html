About NexGo

NexGo is a Next.js-inspired framework built with Go for maximum performance.

Features

  • 🚀 File-based routing (dynamic routes supported)
  • 🔄 Hot reload during development
  • 🌐 SSR and SSG support
  • 🔌 API routes as Go handlers
  • 🎨 Layouts and reusable components
  • ⚡ 50,000+ requests/second
  • 📦 Single binary — zero runtime deps
